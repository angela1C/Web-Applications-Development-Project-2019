Hi, I'm Angela and this is my website for the Web Application Development module submitted as part of the Higher Diploma in Computing and Data Analytics at GMIT.

This website was developed using HTML5, CSS, JavaScript and the JavaScript library D3. I began coding the pages using html elements such as div, paragraph, heading, tables etc and with much reference to W3schools.com. This in turn directed me to trying out using the CSS flex property to structure the pages better.

I did quite a bit of research into using the Flex property and tried out different layouts.
