This is my website for the Web Application Development module submitted as part of the Higher Diploma in Computing and Data Analytics at GMIT.

This website was developed using HTML5, CSS, JavaScript and the JavaScript library D3. It is a personal learning website and not themed around any particular type of business.

The code was written using Visual Studio Code and tested on my web browser Chrome Version 81.0.4044.138 (Official Build) (64-bit). While I had not settled on a final colour scheme and styling, I am going to go with this for now, although I do realise it might not be to everyone's taste. I will revisit though later.

Project Requirements

  • Have a minimum of 3 linked pages
  • Allow the customer to enter their login details and have their login details validated via a login screen
  • Perform Form Validation through JavaScript or HTML to ensure that text fields are not empty and a valid email address is entered.
  • Access and change HTML on the page using the DOM
  • Access and change styling using the DOM
  • Demonstrate the use of events
  • Contain 2 data visualisations, one from a csv file, one from an array

The login details are entered on a page called login.html. The correct login details are angela@gmit.ie and password123

A main external CSS style sheet is used for most of the pages in this website which is located in the css folder. There are some page specific styling including in style tags on the various pages.

To see the visualisations from the CSV files, you must launch a python web server on the folder using python -m http.server.
